About Our Website

We created this website with the intention of allowing a user to add and manage contacts and groups of contacts. We created this website without the use of any frameworks, using pure css, javascript and jquery. Users can create contacts, input the contacts' email address, address, phone number(s), name, gender, date of birth etc. These contacts can also be inserted into named groups so that the user can manage the contacts with a bit more ease. Our website also allows for users to create and leave reviews, change their user and passwords, and view contact info for each member of our team.
